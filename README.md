LogWizard ‚Äì A Simple Log Analysis & Brute-Force Detection Tool
This is a small cybersecurity project I built to understand how log analysis works and how brute-force attacks can be detected using Python.
I created this tool completely from scratch on Windows + VS Code, and I manually made a fake auth.log file to test it.

The main idea of the project is:

Read log lines

Look for failed login attempts

Count how many times an IP failed

If an IP fails 5 or more times ‚Üí mark it as brute-force

Save the alert into a JSON file

This project helped me learn how log files work and how attackers try to access systems.

üîß How I Developed LogWizard (My Step-by-Step Process)
1. Created project folder in Windows

I made a folder named LogWizard and opened it in VS Code.

2. Created a fake log file

Since Windows doesn‚Äôt have /var/log/auth.log, I created my own file called auth.log and added sample failed login entries.

3. Wrote Python code to read the file

My first goal was simply to print the log lines and make sure Python could read them.

4. Added regex to detect failed login attempts

I wrote a regex pattern to extract the IP addresses from log lines containing ‚ÄúFailed password‚Äù.

5. Counted how many times each IP failed

I used a dictionary to store IP ‚Üí count.

6. Added brute-force detection

If an IP had 5 or more failed attempts, I printed a warning on the terminal.

7. Saved the output into a JSON file

Finally, I stored the detected attacks in alerts.json so it looks like a real security report.

This flow helped me understand every part of the logic instead of writing everything at once.
